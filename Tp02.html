Ejercicio 1: enlaces

1.1: URL es uma abreviatura para "Uniform Resource Locator", y sirve para ubicar un recurso de cualquier tipo en el internet, ya sea imagen, video, etc.

1.2: Una URL absoluta proporciona el enlace completo del archivo, este incluye el protocolo (http: // o https: //), dominio y la ruta. Se debe usar una URL absoluta cuando cuando busca un archivo en la web.

1.3: Una URL relativa proporciona unicamente la ruta a un archivo que se relaciona al documento actual. Se debe usar para vincular otro documento en el mismo sitio o servidor.

1.4: La diferencias mas notables de la "url absoluta" y la "url relativa", por nombrar algunas, serian: La "url relativa" proporciona unicamente la ruya en vez del dominio y protocolo, la "url absoluta" se utiliza para buscar un archivo en la web, mientras que la "url relativa" se usa para vincular documentos de un mismo servidor o sitio, etc.

1.10: La etiqueta anchor sirve para crear un enlace a otras paginas de internet, archivos, etc. dentro de la misma pagina.

1.11: El atributo que se utiliza para especificar la URL en la etiqueta anchor, seria el atributo "href"

1.12: Lo que sucede si el atributo "href" esta vacio, es que no se enlaza a ninguna pagina.

1.13: El atributo "target" sirve para especificar el donde desplegar la url enlazada, ya sea mediante un nombre, palabra clave, contexto, etc.

1.14: Un hipervinculo es lo que permite enlazar o conectar paginas web o datos. En html se representa como "a".
